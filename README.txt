To compile and execute : 
 1. mvn clean install 
 2. copy database files into target folder
 3. The database files are case-sensitive and only recognized as: Posts.json, Votes.json, Tags.json
 	Note: They also must follow the same json structure as that of the examples in specs on eclass.
 4. java -jar 291-miniproject2-1.0.SNAPSHOT-jar-with-dependencies.jar <portNums>
